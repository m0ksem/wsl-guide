Find distributive name
wsl.exe --list
Export
wsl.exe --export DistributionName ExportedFileLocation
  • DistributionName — use wsl --list --all to find DistributionName;
  • ExportedFileLocation — file location like work.tar.gz;

Import

wsl.exe --import DistributionName InstallLocation ExportedFileLocation
  • DistributionName — any name for distibution (like Work, Test etc.);
  • InstallLocation — where to install ext4.vhdx, e.g. file system;
  • ExportedFileLocation — location of distor.tar.gz.
Change default user from root to {m0ksem}
Function WSL-SetDefaultUser ($distro="DistroName", $user="UserName") { Get-ItemProperty Registry::HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Lxss\*\ DistributionName | Where-Object -Property DistributionName -eq $distro | Set-ItemProperty -Name DefaultUid -Value ((wsl -d $distro -u $user -e id -u) | Out-String); }; WSL-SetDefaultUser; Remove-Item Function:WSL-SetDefaultUser;
